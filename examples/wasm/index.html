consola-rs — WASM example

Open your browser's developer console (F12) to see styled console output produced by BrowserReporter. Each message uses console.log, console.warn, or console.error with a CSS-styled badge.

Build & serve

# From the repository root:
cd examples/wasm

# Build with wasm-pack
wasm-pack build --target web --release

# Serve this directory
python3 -m http.server 8080
# or: npx serve .

# Open http://localhost:8080 in your browser
